vim configuration for happy java coding

Getting Started

INSTALL

Requirements

  1. jdk 1.8 sudo apt-get install openjdk-8-jdk
  2. Vim 7.3.885+ compiled with lua. If :echo has("lua") returns 1, then you're done; otherwise, see below.

install vim with lua e.g. shell $ sudo apt-get install vim-nox

configration for vim

shell $ mkdir -p ~/.vim/bundle $ git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim $ git clone git@github.com:wsdjeg/DotFiles.git ~/DotFiles $ cp .vimrc vimrc.bak $ cp DotFiles/vimrc .vimrc Launch vim, run :NeoBundleInstall

or

Command run vim +NeoBundleInstall +qall
